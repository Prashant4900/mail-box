# Memory Index

## Product
- [vision] Mailbox is an open-source shared inbox for small teams using Cloudflare Email Routing — no per-seat costs, self-hosted, custom domain email. Read PURPOSE.md for full vision.
- [audience] Indie hackers, SaaS founders, small teams (<20), agencies. Already on Cloudflare, own a custom domain, want to avoid Google Workspace / M365 costs.
- [architecture] Cloudflare receives emails via MX → forwards to Mailbox webhook → processed + stored → shown in shared team inbox UI.
